Social Media API Documentation

This is the documentation for the Social Media API, a RESTful API for a platform.

Users

GET /api/users

Get a list of all users with pagination.

Query Parameters

Parameter Type Description
page Number Page number (default: 1)
limit Number Number of users per page (default: 10)

Response

{
    "success": true,
    "count": 2,
    "page": 1,
    "total_pages": 2,
    "pagination": {
        "next": {
            "page": 2,
            "limit": 10
        }
    },
    "data": [
        {
            "id": "1",
            "username": "traveler",
            "full_name": "Karma",
            "profile_picture": "https://example.com/profiles/traveler.jpg",
            "bio": "Travel photographer",
            "created_at": "2023-01-15"
        },
        ...
    ]
}
GET /api/users/:id

Get a specific user by ID.

Parameters

Parameter Type Description
id String User ID

Response

{
    "success": true,
    "data": {
        "id": "1",
        "username": "traveler",
        "full_name": "Karma",
        "profile_picture": "https://example.com/profiles/traveler.jpg",
        "bio": "Travel photographer",
        "created_at": "2023-01-15"
    }
}
POST /api/users

Create a new user.

Request Body

{
    "username": "new_traveler",
    "email": "new@example.com",
    "password": "securepassword",
    "full_name": "New Traveler",
    "bio": "Adventure seeker"
}

Response

{
    "success": true,
    "data": {
        "id": "4",
        "username": "new_traveler",
        "full_name": "New Traveler",
        "email": "new@example.com",
        "profile_picture": "default-profile.jpg",
        "bio": "Adventure seeker",
        "created_at": "2023-03-20"
    }
}
PUT /api/users/:id

Update an existing user.

Parameters

Parameter Type Description
id String User ID

Request Body

{
    "username": "updated_traveler",
    "full_name": "Updated Traveler",
    "bio": "Updated bio"
}

Response

{
    "success": true,
    "data": {
        "id": "1",
        "username": "updated_traveler",
        "full_name": "Updated Traveler",
        "profile_picture": "https://example.com/profiles/traveler.jpg",
        "bio": "Updated bio",
        "created_at": "2023-01-15"
    }
}
DELETE /api/users/:id

Delete a user by ID.

Parameters

Parameter Type Description
id String User ID

Response

{
    "success": true,
    "data": {}
}
